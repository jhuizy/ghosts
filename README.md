# Ghosts

## Intro

A demonstration of the [Ghosts of departed proofs](https://kataskeue.com/gdp.pdf) in a simple web app

## Development

```bash
git clone https://github.com/jhuizy/ghosts 
cd ghosts 
nix-shell
```
